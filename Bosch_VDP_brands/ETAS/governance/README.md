# Governance

`RoEAIFile.md` remains at the workspace root so agents and humans see it first.

Use this directory for supplemental governance artifacts only, such as:

- Program rule snapshots or links.
- Scope decision records.
- Human approval notes for Class 2 actions.
- Stop-condition records.

Do not move `RoEAIFile.md` here unless future automation is updated to read the new path.
