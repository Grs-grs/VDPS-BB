# Artifacts

Store only indexes or redacted references here by default.

Recommended subfolders, created only when needed:

- `headers/`: redacted response headers.
- `screenshots/`: redacted screenshots.
- `html-summaries/`: notes about saved HTML, not raw sensitive content.
- `request-logs/`: minimal request accounting.

Large raw files, secrets, credentials, session cookies, personal data, and customer data must not be stored here.
