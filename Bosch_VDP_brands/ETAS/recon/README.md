# Recon

Passive recon and classification files live here.

## Files

- `target_candidates.md`: ranked passive target candidates and hypotheses.
- `domains.md`: CT and public hostname inventory.
- `docs.md`: public documentation and download pivots.
- `apps.md`: app-store and package-ID notes.
- `passive_sitemap_inventory.md`: sitemap-derived product/version inventory.

Keep active tests and vulnerability claims out of this directory. Promote validated observations into `findings/`.
