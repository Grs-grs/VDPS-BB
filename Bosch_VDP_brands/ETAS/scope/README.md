# Scope

This directory separates active host allowlisting from passive leads.

## Files

- `current-scope.md`: preserved content from the original flat `scope` file.
- `allowed-hosts.txt`: exact hostnames currently allowed for low-impact public observation under the recorded operator approval and RoE.
- `approval-required-hosts.txt`: hosts and services that require explicit human approval before interaction.

## Rule

Being listed in passive recon is not enough for active testing. Record exact scope confirmation and request budget before any live validation.
